WORKFLOW:
1. User selects AI provider (Cloudflare/OpenAI/Google)
2. User enters API key (stored in localStorage)
3. User customizes prompt template if needed
4. User clicks "Test AI" to verify configuration
5. User clicks "Save Settings" to persist
6. Settings are used automatically when user clicks "AI Assistant" 
   button in product editor

## Key Features

### Multi-Provider Support
- ✅ Cloudflare AI (Llama 3.1 70B)
- ✅ OpenAI (GPT-4/GPT-3.5)
- ✅ Google AI (Gemini Pro)

### Security
- 🔒 API keys stored only in browser localStorage
- 🔒 Never sent to or stored on server
- 🔒 Settings validated on server side
- 🔒 Encrypted connections (HTTPS)

### Customization
- ⚙️ Configurable AI provider
- ⚙️ Custom model selection
- ⚙️ Adjustable temperature (creativity)
- ⚙️ Custom max tokens
- ⚙️ Fully customizable prompt template

### User Experience
- 💾 One-click save
- 🧪 Test before using
- 🔄 Reset to defaults
- 📝 Clear instructions
- ℹ️ Helpful tooltips
