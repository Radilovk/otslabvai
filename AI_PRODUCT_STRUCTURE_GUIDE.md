# Структура на продуктова информация за AI Асистент

## Въведение

Този документ описва очакваната структура и начин на въвеждане на продуктова информация от AI асистента в админ панела. Базиран е на анализ на примерни продукти: **Thermo Master**, **Lida Green** и **MeiziMax**.

## Обща структура на продукт

Всеки продукт се състои от две основни секции:
1. **`public_data`** - Публично видима информация за крайните потребители
2. **`system_data`** - Системна и административна информация

---

## 1. PUBLIC_DATA - Публична информация

### 1.1 Основна информация за продукта

```json
{
  "name": "MeiziMax",
  "tagline": "Японска формула за стройна фигура и сияйна кожа",
  "price": 38.35,
  "description": "MeiziMax е елегантният избор за отслабване, вдъхновен от японската философия за хармония между тяло и дух. Това е вторият по сила продукт в каталога, но с уникален разкрасяващ ефект...",
  "image_url": "/images/products/meizimax.png"
}
```

**Изисквания:**
- **name** (задължително): Кратко, запомнящо се име на продукта
- **tagline** (задължително): Кратко маркетингово послание (30-60 символа)
- **price** (задължително): Число с точност до 2 десетични знака
- **description** (задължително): Подробно описание (200-400 думи), емоционално и убедително
- **image_url** (опционално): Път до изображението на продукта

### 1.2 Научна легитимност

```json
{
  "research_note": {
    "text": "Усъвършенствана формула за детокс и отслабване",
    "url": "https://pubmed.ncbi.nlm.nih.gov/?term=japanese+herbal+weight+loss"
  }
}
```

**Изисквания:**
- **text**: Кратко твърдение за научна основа или качество
- **url**: Линк към PubMed, научно изследване или "#" ако няма конкретен източник

### 1.3 Ефекти на продукта

```json
{
  "effects": [
    {
      "label": "Изгаряне на мазнини",
      "value": 85
    },
    {
      "label": "Потискане на апетита",
      "value": 90
    },
    {
      "label": "Детокс и пречистване",
      "value": 90
    }
  ]
}
```

**Изисквания:**
- Масив от **3 ефекта** (точно 3, не повече, не по-малко)
- **label**: Името на ефекта на български език
- **value**: Число от 0 до 100, представящо силата на ефекта
  - **70-80**: Умерен ефект (примерно Thermo Master)
  - **85-95**: Силен ефект (примерно MeiziMax, Lida Green)
  - **95-100**: Много силен/максимален ефект (примерно Lida Green)

- **url**: Линк към поръчка (обикновено "#" в момента)

## 2. SYSTEM_DATA - Системна информация

### 2.1 Основни системни данни

```json
{
  "application_type": "Oral / Capsules",
  "inventory": 30,
  "goals": [
    "weight-loss",
    "appetite-control",
    "detox"
  ],
  "target_profile": "Жени, търсещи силно отслабване комбинирано с разкрасяващ ефект."
}
```

**Изисквания:**
- **application_type**: Начин на приложение
  - Примери: "Oral / Capsules", "Oral / Tablets", "Oral / Liquid", "Oral / Powder"
- **inventory**: Наличност на склад (цяло число)
- **goals**: Масив от цели (keyword формат с тирета)
  - Примери: "weight-loss", "appetite-control", "detox", "energy", "natural", "wellness"
- **target_profile**: Описание на целева аудитория (1-2 изречения)

### 2.2 Протокол за прием

```json
{
  "protocol_hint": "ПРИЕМ: 1 капсула сутрин (преди 9:00) на празен стомах с 2 чаши вода. ПРОДЪЛЖИТЕЛНОСТ: Курс от 1-3 месеца за оптимални резултати. Може да се прави почивка от 2 седмици между курсовете. ВАЖНО: Задължително пийте поне 2.5 литра вода дневно за детокс ефект."
}
```

**Изисквания:**
- Структуриран текст с 3 основни части:
  1. **ПРИЕМ**: Точно кога, как и колко да се приема
  2. **ПРОДЪЛЖИТЕЛНОСТ**: Препоръчителна продължителност на курса
  3. **ВАЖНО**: Допълнителни критични бележки

### 2.3 Предупреждения за безопасност

```json
{
  "safety_warnings": "ПРОТИВОПОКАЗАНИЯ: Не се препоръчва при бременност, кърмене, сърдечно-съдови проблеми, високо кръвно налягане, диабет. Не е подходящ за лица под 18 години. ВНИМАНИЕ: Може да причини сухота в устата, повишена жажда. Пийте поне 2.5 литра вода дневно. Не превишавайте препоръчителната доза от 1 капсула дневно."
}
```

**Изисквания:**
- Структуриран текст с 2 основни части:
  1. **ПРОТИВОПОКАЗАНИЯ**: Кой НЕ трябва да приема продукта
  2. **ВНИМАНИЕ**: Възможни странични ефекти и предпазни мерки

### 2.4 Синергични продукти

```json
{
  "synergy_products": []
}
```

**Изисквания:**
- Масив от product_id на продукти, които работят добре в комбинация
- Обикновено празен масив `[]`, освен ако няма конкретни препоръки

## 4. Най-добри практики за AI генериране

### 4.1 Тон на гласа

- **Директен и уверен**: Избягвайте несигурност
- **Емоционален**: Свържете продукта с желанията на клиента
- **Професионален**: Балансирайте маркетинг с факти
- **На български език**: Естествен, разговорен български

### 4.2 Структура на текстовете

- **Кратки параграфи**: 2-4 изречения
- **Активен глас**: "Продуктът ускорява метаболизма" вместо "Метаболизмът се ускорява"
- **Конкретни обещания**: "След 2 седмици" вместо "Скоро"
- **Ползи преди характеристики**: Какво печели клиентът, не просто какво съдържа

### 4.3 Маркетингов език

#### За леки продукти (Thermo Master):
- "нежно", "балансирано", "природно", "подкрепа", "хармония"

#### За силни продукти (MeiziMax):
- "ефективно", "резултати", "трансформация", "интензивно"

#### За много силни продукти (Lida Green):
- "максимално", "безкомпромисно", "пълен контрол", "драстично", "видимо"

### 4.4 Предупреждения и честност

- **Винаги включвайте противопоказания**
- **Споменавайте възможни странични ефекти**
- **Не обещавайте невъзможното**
- **Балансирайте маркетинг с отговорност**
