# LIFE BioHack — Design Brief
## Визуална идентичност: Фармацевтична Чистота × Висока Технология × Биотехнология

---

## 🎯 Концепция

**LIFE BioHack** представя себе си като научно-ориентиран биохак бранд на пресечната
точка между фармацевтичната прецизност, авангардните биотехнологии и съвременния хайтек.

Визуалното усещане трябва да предизвиква:
- **Фармацевтична чистота** — чисти бели фонове, клинична точност, стерилна прецизност
- **Наука и изследвания** — лабораторна естетика, технически шрифтове, данни и графики
- **Висока технология** — прецизни линии, монохромни дълбини, цифрови детайли
- **Биотехнология** — молекулярни мотиви, ДНК структури, хексагонални мрежи, циан (teal) акценти

| Вторичен фон            | Студено бяло      | `#F7F9FB`  | Секции/карти — студено, не топло              |
| Фин акцент фон          | Лед               | `#EEF3F8`  | Разграничения, hover состояния                |
| Основен акцент          | Биотех циан       | `#00C4C8`  | CTA, highlight, икони, линии                  |
| Тъмен акцент (hovер)    | Дълбок циан       | `#0099A8`  | Hover на CTA бутони                           |
| Премиум (вторичен)      | Научно злато      | `#D4A843`  | Бадж "premium", граничен акцент, цени         |
| Основен текст           | Тъмно морско      | `#0F2540`  | Заглавия, тяло (тъмно, но не черно)           |
| Вторичен текст          | Синкаво-сиво      | `#4A6A85`  | Описания, подзаглавия                         |
| Граница                 | Прозрачен циан    | `rgba(0,196,200,0.15)` | Карти, секции, разделители    |

домашна/козметична естетика. Те са допустими единствено в малки баджове или специфични
UI елементи с ясна цел.

## 🔤 Типография

| Роля                  | Шрифт                     | Вариант        |
|-----------------------|---------------------------|----------------|
| Дисплей заглавия (h1) | **Cormorant Garamond**    | 300–600 / Italic |
| UI заглавия (h2–h4)   | **Space Grotesk**         | 500–700        |
| Тяло текст            | **DM Sans**               | 400–500        |
| Моноширочен (код/данни)| **Courier New** / `monospace` | 400          |

**Принципи:**
- H1 и дисплей заглавия — елегантна serif, внушава доверие и heritage
- UI елементи — геометричен sans-serif за четливост и технически вид
- Тяло — неутрален, четлив sans-serif
- Важни числа/научни данни — monospace за автентичност

## 🧪 Дизайн Принципи

### 1. Чистота като стандарт
- Белият фон е основното "платно" — като чиста лаборатория
- Минимален хроматичен шум: ограничени цветни блокове
- Голям негативен пространство (whitespace)

### 2. Научна Прецизност
- Хексагонални мотиви в фоновете (молекулярни решетки)
- Тънки прецизни граници (1px, не 2px) с ниска непрозрачност
- Числа и статистики показани с научен стил (моноширочен шрифт)

### 3. Технологичен Авторитет
- Субтилни градиенти само в посока: чисто бяло → студено светло сиво
- Акцентен циан (#00C4C8) — "биотех тил", напомня на кюветки/реагенти
- Сенките са нежни (0px 4px 24px rgba(0,0,0,0.06)) — никога тежки

### 4. Прозрачност и Доверие
- Карти с бял фон и тънка цианова граница
- Без прекомерни декорации — всеки елемент има функция
- Ясна визуална йерархия

## 🏗 Компонент Наръчник

### Фонове на секции
```
Hero:               бял → студено сиво (#F7F9FB) — 155° diagonal, субтилен hex pattern
Продукти:           чисто бяло (#FFFFFF)
Алтернативни:       студено сиво (#F7F9FB)
Footer:             студено сиво (#F7F9FB) с тънка цианова граница
```

### Карти
```
Фон:        #FFFFFF (чисто бяло)
Граница:    1px solid rgba(0, 196, 200, 0.15)
Сянка:      0 4px 20px rgba(0,0,0,0.06)
Hover:      border-color → #00C4C8 + box-shadow → rgba(0,196,200,0.15)
Radius:     14px (прецизно закръглен, не overrounded)
```

### CTA Бутони
```
Primary:    background: linear-gradient(135deg, #00C4C8 0%, #00A8AC 100%)
            padding: 14px 32px
            border-radius: 8px (по-прецизен от 14px)
            font-weight: 700, uppercase, letter-spacing: 0.06em
Secondary:  бял фон + 1.5px #00C4C8 граница + #0F2540 текст
```

### Promo Banner
```
Фон:        чисто бяло
Текст:      #0F2540 (тъмно морско) + #00C4C8 акценти
Граница:    solid 1px rgba(0, 196, 200, 0.25) отдолу
```

## 🔬 Вдъхновение

- **Pharma packaging**: Genentech, Roche, BioNTech — чисто бяло + прецизни тилови/сини акценти
- **Med-tech UI**: Apple Health, Fitbit Sense — чисти карти, голям whitespace
- **Biotech branding**: Moderna, CRISPR Therapeutics — синьо-тилова палитра, технически

---

## 📐 Spacing & Layout

- **Секции**: `padding: 80px 0` (desktop), `40px 0` (mobile)
- **Карти**: `padding: 28–32px`
- **Grid gap**: `28–32px` между карти
- **Container**: `max-width: 1200px`, центрирани
- **Мобилно**: single column, с `padding: 0 16–20px`
